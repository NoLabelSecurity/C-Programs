# CPT234 C Programs

This repository contains a collection of C programming assignments and practice programs completed for the **CPT234 course**. The purpose of this repo is to demonstrate understanding of core C programming concepts, problem-solving skills, and foundational computer science principles.

---

## 📘 Course Overview

CPT234 focuses on developing proficiency in the C programming language, with an emphasis on:

- Writing efficient and structured code
- Understanding how memory works at a low level
- Building logical problem-solving skills
- Preparing for more advanced topics like systems programming and cybersecurity

---

## 🧠 Topics Covered

The programs in this repository cover a range of key concepts, including:

### 🔹 Basics
- Variables and data types
- Operators and expressions
- Input and output (`printf`, `scanf`)

### 🔹 Control Structures
- Conditional statements (`if`, `else`, `switch`)
- Loops (`for`, `while`, `do-while`)

### 🔹 Functions
- Function creation and usage
- Parameter passing
- Return values

### 🔹 Arrays and Strings
- One-dimensional and multi-dimensional arrays
- String manipulation using standard libraries

### 🔹 Pointers
- Pointer fundamentals
- Pointer arithmetic
- Relationship between arrays and pointers

### 🔹 File Handling
- Reading from and writing to files
- File pointers and modes

### 🔹 Structures
- Defining and using `struct`
- Organizing related data
